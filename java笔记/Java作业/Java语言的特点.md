**机器码：**

**机器码指的是软件根据计算机的硬件信息,例如CPU、内存、主板序列号等,按照一定的算法生成的一串无规律的字符串,并且在不同计算机上生成的机器码是不一样的,因为每台计算机的硬件信息不一样。**

**Java中，在JVM上跑的是字节码文件，字节码文件必须经过进一步的处理才会变为机器码。**
